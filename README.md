# Docs_Search

A Python-based documentation search tool designed to help you quickly find and retrieve information from documentation files.

## Features

- 🔍 **Fast Search** - Efficiently search through documentation
- 📚 **Multi-format Support** - Works with various documentation formats
- 🎯 **Relevant Results** - Returns the most relevant search results
- 💻 **Python-based** - Built with Python for ease of use and customization

## Getting Started

### Prerequisites

- Python 3.7 or higher
- pip (Python package manager)

### Installation

```bash
git clone https://github.com/RaviKumarDube/Docs_Search.git
cd Docs_Search
pip install -r requirements.txt
```

### Usage

```python
# Basic example
from docs_search import search

results = search("your search query", docs_path="./docs")
for result in results:
    print(result)
```
